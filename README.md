# Dice!
QQ Dice Robot For TRPG Based on CoolQ 

## 简介

旧日的儿子是一款基于酷Q的QQ跑团掷骰机器人 主人是旧日酱（2558272027）

基于溯洄的Dice项目(https://github.com/w4123/Dice)进行更改
